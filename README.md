# tttermwidget
Terminal Emulator used in theTerminal

tttermwidget is derived from the work of [QTermWidget](https://github.com/lxqt/qtermwidget).

## Dependencies
- qmake
- [the-libs](https://github.com/vicr123/the-libs)

## Build
```
qmake
make
```

## Install
```
make install
```
